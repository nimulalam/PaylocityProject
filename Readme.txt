This project is covering Deleting all existing employees, adding a new employee and updating the same employee.
The framework that I am using is TestNG annotations along with reusable methods which has Try and Catch and explicit wait.

To open in IDE
-- download intellij community edition IDE if needed
   https://www.jetbrains.com/idea/download/#section=mac
-- open existing project as Maven project

To execute the test case do the following:
--Go to src -> test -> java -> Paylocity_Test_Scenarios.java
--Right click anywhere out side of the class and click Run

To view the reusable methods:
--Go to src -> main -> java -> Reusable_Library

To view HTML Report:
--Go to src -> main -> java -> HTML_Report
--Right click on the Automation_Report.html
--Open in any browser
